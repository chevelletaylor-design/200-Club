# Asset Guide — The 200 Club of Monmouth County

This guide tells you exactly how to name, size, and orient photos before uploading them to the site.

## 🎯 The Golden Rule

**Two portrait slots. Everything else landscape.** The site is built for landscape event photography everywhere except two specific headshot slots.

### Portrait slots (4:5 ratio — tall):
- 🏠 Home → Featured Story → Officer Russo (or whichever Valor recipient is featured)
- 🏠 Home → President's Message → Mary Pat Angelini (also appears on About page)

### Landscape slots (3:2 or 16:9 ratio — wide):
- 🏠 Home → hero slideshow (full-bleed background)
- 🎓 Programs → Scholarship carousel
- 🎓 Programs → Valor Awards carousel
- 🎓 Programs → Spotlight 200 story photos
- 📰 Media → Gallery cover thumbnails
- 📰 Media → YouTube videos (auto-pulled from YouTube)

## 📸 Photo Upload Method

**No GitHub needed.** All photos go up through the admin panel at **`[SITE URL]/admin/`**.

Click any 📷 image field, drag your file in, click Publish. Photo is live within 60 seconds.

## 🏛️ The Logo

The official 200 Club crest lives in the header and footer on every page. To upload it:

1. Save the logo as `logo.png` (with transparent background if possible) or `logo.svg`
2. Admin panel → 🏷️ Brand & Logo → Logo image → upload
3. Publish

Until the official crest is uploaded, a built-in default (navy shield + "200 CLUB" + "WE REMEMBER") displays in its place.

**Recommended logo specs:**
- Square or near-square aspect ratio
- 300×300px minimum, 800×800px ideal
- PNG with transparent background OR SVG

## ✏️ Filename Rules

Always use filenames that are:
- ✅ **All lowercase** (`hero-1.jpg` not `Hero 1.JPG`)
- ✅ **No spaces** (use hyphens: `mary-pat-angelini.jpg`)
- ✅ **No special characters** (no apostrophes, ampersands, exclamation marks)
- ✅ **Standard file types** (`.jpg` for photos, `.png` for logos/QR, `.svg` for vector logos)

Good: `hero-1.jpg`, `officer-russo.jpg`, `gal-2024-valor.jpg`
Bad: `Hero Photo #1.JPG`, `Mary Pat's Photo.jpeg`

## 📏 Photo Size & Compression

Large photos slow down mobile, which hurts both UX and Google rankings.

**Before uploading any photo:**
1. **Check orientation** — match the required ratio (portrait 4:5 for the two headshot slots, landscape 3:2 or 16:9 for everything else)
2. **Resize** — Open in Preview (Mac) → Tools → Adjust Size → width 1600–1920px
3. **Compress** — Drop into [tinypng.com](https://tinypng.com) (free)
4. Aim for files under 500KB each

## 🆘 Troubleshooting

**"My portrait photo looks stretched"** → You uploaded a landscape photo in a portrait slot, or vice-versa. Check the section in this guide — the two portrait slots are clearly marked.

**"My photo is blurry"** → File was too small. Re-upload at 1600px+ width.

**"My photo is sideways"** → Phone photos sometimes have wrong rotation data. Open in Preview → rotate → save → re-upload.

**"My photo uploaded but doesn't show"** → Wait 60 seconds for the site to rebuild. Hard refresh (Cmd+Shift+R or Ctrl+Shift+R on Windows).

**"Wrong file uploaded by mistake"** → Just upload the correct file with the same filename — it replaces the old one.

**"A gallery tile is missing its cover photo"** → The gallery card defaults to a 📷 placeholder tile with the filename shown. Upload a cover image via 📰 MEDIA PAGE → Photo Galleries → (gallery) → Cover image.
