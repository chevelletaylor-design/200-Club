# Asset Guide — The 200 Club of Monmouth County

This guide tells you exactly how to name and size your photos before uploading them to your site.

## 📸 Photo Upload Method

**You don't need GitHub.** All photos go up through the admin panel at **`[SITE URL]/admin/`**.

Just click any 📷 image field in the admin panel, drag your file in, and click Publish. The photo is live within 60 seconds.

## ✏️ Filename Rules

Always use filenames that are:
- ✅ **All lowercase** (`sophia.jpg` not `Sophia.JPG`)
- ✅ **No spaces** (use hyphens: `sophia-taylor.jpg`)
- ✅ **No special characters** (no apostrophes, ampersands, exclamation marks)
- ✅ **Standard file types** (`.jpg` for photos, `.svg` or `.png` for logos/QR)

Examples:
- ✅ Good: `hero-1.jpg`, `officer-russo.jpg`, `paypal-qr.png`
- ❌ Bad: `Hero Photo #1.JPG`, `Officer Russo's Photo.jpeg`

## 📏 Photo Size & Compression

Large photos slow down your site on mobile, which hurts user experience and Google rankings.

**Before uploading any photo:**
1. **Resize** — Open in Preview (Mac) → Tools → Adjust Size → set width to 1600px or less
2. **Compress** — Drop the file into [tinypng.com](https://tinypng.com) (free)
3. Aim for files under 500KB each

## 🔄 Replacing the Default Logo

The site comes with a built-in 200 Club seal (navy + gold). If you want to upload the official 200 Club logo:

1. Save logo as `logo.svg` (preferred) or `logo.png`
2. Admin panel → 🏷️ Brand & Logo → click the Logo upload field
3. Upload your file
4. Publish

The logo will automatically appear in the header and footer.

## 🆘 Troubleshooting

**"My photo is blurry"** → Uploaded file was too small. Re-upload at 1600px width minimum.

**"My photo is sideways"** → Phone photos sometimes have wrong rotation data. Open in Preview (Mac) → rotate → save → re-upload.

**"My photo uploaded but doesn't show"** → Wait 60 seconds for the site to rebuild. Hard refresh (Cmd+Shift+R or Ctrl+Shift+R).

**"Wrong file uploaded by mistake"** → Just upload the correct file again with the same filename — it replaces the old one.
